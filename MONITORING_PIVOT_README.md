# تغییر مسیر پروژه: از «پیش‌بینی» به «مانیتورینگ صادقانه»

## چرا این تغییر؟
بعد از تست‌های آماری دقیق (`edge_diagnostic.py`, `triple_barrier_diagnostic.py`)
مشخص شد که اندیکاتورهای تکنیکال معمولی (RSI, MACD, Bollinger, ATR, SMA/EMA)
روی BTC/USDT در تایم‌فریم ۴ساعته **هیچ قدرت پیش‌بینی معنی‌داری** ندارند
(AUC ≈ 0.50 در همه‌ی آزمایش‌ها). این نتیجه با دو نوع لیبل‌گذاری متفاوت
(افق‌ثابت و triple-barrier) تکرار شد، پس مشکل از نوع مدل یا لیبل نبوده.

به‌جای ادامه‌ی تنظیم پارامترهای یک مدل که پایه‌ی آماری نداره، پروژه رو به
یک ابزار **صادق و واقعاً قابل‌استفاده** تغییر دادیم: به‌جای «پیش‌بینی»،
حالا فقط **شرایط قابل‌توجه بازار رو با قوانین شفاف رصد و گزارش می‌کنیم**.

## فایل جدید اصلی: `monitor.py`
جایگزین `main.py` قدیمی. منطقش کاملاً متفاوته:
- هیچ مدل ML یا احتمال "X% مطمئنم" وجود نداره
- فقط قوانین ساده و قابل‌فهم چک می‌شن (تعریف‌شده در `alert_rules.py`)
- هر هشدار توضیح می‌ده *چرا* فرستاده شده (مثلاً "RSI به ۲۸ رسیده")
- هیچ هشداری ادعای جهت آینده‌ی قیمت نمی‌کنه

## قوانین فعلی (در `alert_rules.py`)
1. RSI در منطقه‌ی اشباع خرید/فروش
2. برخورد قیمت به باندهای بولینگر
3. کراس‌اوور MACD
4. جهش غیرعادی حجم معاملات
5. کراس‌اوور میانگین متحرک (کراس طلایی/مرگ کوتاه‌مدت)
6. افزایش شدید نوسان بازار (ATR)

می‌تونی به‌راحتی قانون جدید اضافه کنی: یک تابع جدید در `alert_rules.py`
بنویس که یک دیکشنری `{"name":..., "detail":...}` برمی‌گردونه، و در
`evaluate_all_rules` صداش کن.

## اجرا
```bash
python monitor.py
```
همون تنظیمات `config.py` (SYMBOL, TIMEFRAME, TELEGRAM_*) استفاده می‌شه.

## فایل‌های قدیمی (ML) چیکار کنیم؟
پیشنهاد می‌کنم `main.py`, `model_train.py`, `model.joblib` و بک‌تست‌های
مرتبط رو **حذف نکنی** بلکه در یک پوشه‌ی جدا (مثلاً `archive_ml_experiment/`)
نگه داری - هم به‌عنوان یادگاری از یک تحلیل آماری درست و کامل، هم برای
اینکه اگه بعداً فیچرهای واقعاً جدید (مثل funding rate یا open interest)
اضافه کردی، بتونی همون پایپ‌لاین رو دوباره استفاده کنی.

## یک نکته‌ی صادقانه‌ی دیگه
این ابزار جدید هم "استراتژی سودده" نیست - هیچ‌وقت ادعا نمی‌کنه که هست.
فقط بهت کمک می‌کنه بدون نیاز به چک کردن مداوم چارت، از اتفاقات قابل‌توجه
بازار باخبر بشی. تصمیم معامله و مدیریت ریسک همیشه با خودته.
