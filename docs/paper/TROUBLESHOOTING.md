# عیب‌یابی

- `BLOCKED_RESEARCH_PAPER`: blockerهای preflight را رفع کنید؛ داده یا artifact اجباری را جعل نکنید.
- `BLOCKED_STRATEGY_PAPER`: شواهد آماری یا Fresh OOS کافی نیست؛ threshold را روی Holdout تغییر ندهید.
- `NO_ELIGIBLE_NEW_OBSERVATIONS`: چرخه سالم بوده ولی فرصت تازه واجد شرایط نبوده است.
- `PAPER_STATE_NOT_WRITABLE`: دسترسی پوشه `logs/paper_launch_v2` را اصلاح کنید.
- خطای Telegram: token/chat id را در `.env` یا GitHub Secrets بررسی کنید؛ کلید معاملاتی لازم نیست.
