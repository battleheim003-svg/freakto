# Freakto v5.1.0 — Forward Test Data Collector

## Purpose

v5.0 added Portfolio Memory, Confidence Calibration, and Monte Carlo Risk Lab. The next correct step is not live trading; it is controlled forward data collection.

This release adds a safe Forward Test Controller that runs the paper-only data collection workflow and tracks progress toward practical testing requirements.

## Added

- `engine/forward_test.py`
- `forward_test_dashboard.py`
- `FORWARD_TEST_RUNBOOK.md`
- `run_forward_test_cycle.bat` generated by `--write-bat`
- `run_forward_test_status.bat` generated by `--write-bat`

## Main commands

```cmd
python forward_test_dashboard.py
python forward_test_dashboard.py --plan
python forward_test_dashboard.py --cycle --validate --continue-on-error
python forward_test_dashboard.py --cycle --validate --continue-on-error --send
python forward_test_dashboard.py --write-bat
```

## What the cycle runs

1. `python monitor.py --once`
2. `python portfolio_scanner.py --paper`
3. `python decision_evaluator.py`
4. `python paper_trading_dashboard.py --evaluate`
5. Optional: `python validation_suite_dashboard.py`
6. Optional: `python risk_lab_dashboard.py`

## Tracked targets

- Complete evaluations: 100
- Closed paper trades: 30
- Regime-labeled samples: 30
- Forward-test days: 30

## Safety

No live trading, no exchange order placement, no API trading action.
This is only a data collection and validation workflow.
