# Empirical Calibration + Edge Gate

این مرحله Score خام موتور را حذف نمی‌کند؛ بلکه قبل از `ACTIONABLE` شدن، آن را با نتیجه‌های واقعی Replay کالیبره می‌کند.

## فایل ورودی

مسیر پیش‌فرض:

```text
logs/calibration/score_calibration.csv
```

حداقل ستون‌های لازم (نام‌های هم‌معنی نیز پشتیبانی می‌شوند):

```csv
raw_score,observed_success_rate,sample_count
50,0.44,620
60,0.49,590
70,0.56,510
80,0.63,390
90,0.71,240
```

`observed_success_rate` می‌تواند بین 0 و 1 یا به‌صورت درصد 0 تا 100 باشد.

## رفتار ایمن

- نبودن یا خراب بودن فایل، موتور را متوقف نمی‌کند.
- بدون کالیبراسیون معتبر، `Edge Gate` بسته می‌ماند.
- Score بالا به‌تنهایی دیگر برای `ACTIONABLE` کافی نیست.
- حداقل پیش‌فرض: 100 نمونه، احتمال موفقیت 55٪ و Edge حداقل 3٪ بالاتر از نقطه سربه‌سر 50٪.
- نگاشت بین bucketها با interpolation انجام می‌شود و حجم نمونه ضعیف‌ترِ دو bucket ملاک قرار می‌گیرد.
