# Cleanup Notes

این نسخه برای ادامه‌ی توسعه روی مسیر **Market Monitoring** آماده شده است.

## تغییرات انجام‌شده

- حذف کلیدهای hardcode از کد و انتقال تنظیمات حساس به `.env`.
- افزودن `.env.example` برای راه‌اندازی امن پروژه.
- اصلاح `telegram_notifier.py` که قبلاً token و chat id را مستقیم داخل URL/Data گذاشته بود.
- اصلاح `data_fetcher.py` تا کلیدهای WunderTrading را فقط از `config.py` بخواند.
- حذف `__pycache__` و افزودن `.gitignore`.
- انتقال فایل‌های آزمایشی، بک‌تست، ML و نسخه‌های قدیمی به پوشه‌ی `archive/`.

## مسیر پیشنهادی اجرا

```bash
cp .env.example .env
# مقدارهای واقعی را داخل .env وارد کن
python monitor.py
```

## نکته‌ی امنیتی مهم

کلیدهایی که قبلاً داخل پروژه بودند باید حتماً در سرویس‌های مربوطه revoke/rotate شوند. پاک‌کردن از فایل کافی نیست، چون ممکن است قبلاً در history، zip، لاگ یا سیستم‌های دیگر ذخیره شده باشند.
